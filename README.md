# Data Science & Inteligência Geoespacial — Portfólio Técnico

Repositório central de projetos em **Ciência de Dados** com foco em **Inteligência Geoespacial**:
análise estatística, consumo de APIs públicas e processamento de grandes bases territoriais.

---

## Projetos

| Projeto | Descrição | Stack |
|---------|-----------|-------|
| [Monitoramento NDVI — Ceará (2005–2024)](./projeto-ndvi) | Análise multitemporal da Caatinga via GEE. Correlação entre NDVI e precipitação acumulada (FUNCEME) em 184 municípios. Variação de +36,8% no vigor vegetativo. | Python · Rasterio · API GEE |
| [Plano Diretor de Fortaleza — IPPLAN](./plano-diretor-fortaleza) | Cartografia temática oficial e indicadores de resiliência ambiental urbana para o macrozoneamento de Fortaleza. Anexos págs. 183–189 da Minuta do PDPFor. | Python · Geopandas · GEE · QGIS |
| [Pharma Intelligence BR](./pharma-intelligence) | BI territorial full-stack para o varejo farmacêutico. Cruzamento IBGE × CNES para mapeamento de regiões com alta renda e baixa saturação de mercado. | Python · FastAPI · JavaScript |
| [PIB Municipal — Ceará](./analise_pib_mun_ce) | Evolução econômica municipal com coleta automatizada via API SIDRA/IBGE. Análise de disparidades regionais. | Python · Pandas · API IBGE |
| [Diagnóstico Agropecuário — MG](./analise_agro_mg) | Produtividade e cadeias produtivas regionais com pipeline ETL via conexão direta em SQL. | Python · SQL |
